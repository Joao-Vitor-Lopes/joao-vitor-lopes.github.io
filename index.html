Conhecimentos

Html

HTML é a linguagem usada para estruturar o conteúdo das páginas web, organizando textos, imagens, links e elementos com tags semânticas.

Css

CSS é responsável por estilizar o conteúdo HTML, definindo cores, fontes, espaçamentos e tornando as páginas responsivas e visuais.

Javascript

JavaScript é a linguagem que traz interatividade para as páginas web, permitindo animações, validações de formulários e comunicação com servidores em tempo real.

Git

Git é um sistema de controle de versão que permite rastrear alterações no código, colaborar com outros desenvolvedores e manter o histórico de desenvolvimento de projetos.

Linguagem C

A linguagem C é uma linguagem de programação estruturada, muito usada em sistemas embarcados, sistemas operacionais e aplicações que exigem alto desempenho.

Figma

Figma é uma ferramenta de design colaborativo para criação de interfaces, protótipos e layouts, ideal para designers e desenvolvedores trabalharem em conjunto.

SQL

SQL é a linguagem usada para gerenciar e consultar bancos de dados relacionais, permitindo inserir, buscar e organizar informações.

Python

Python é uma linguagem de programação versátil, com sintaxe simples e clara, usada em automação, análise de dados, desenvolvimento web, IA e muito mais.

Links Lopes

Este é um projeto de site que criei para centralizar todos os meus links de redes sociais em um só lugar. A ideia é tornar o acesso aos meus perfis mais fácil e rápido, oferecendo uma maneira prática de compartilhá-los de forma organizada. O layout é simples e intuitivo, com uma navegação fácil, para que qualquer pessoa consiga acessar meus links com um clique. Esse site me ajuda a reunir todas as minhas redes sociais em um único ponto, facilitando a forma como compartilho minha presença online.

Sapo Utopia

Em um projeto institucional, criei um site dedicado aos sapos, utilizando uma certa ferramenta de pesquisa de pesquisa. O objetivo era proporcionar uma experiência informativa e envolvente, permitindo aos usuários explorar diversas facetas desses anfíbios. O site apresentou uma interface atrativa, integrando informações detalhadas sobre espécies, habitats e curiosidades. Transformando o site em uma valiosa fonte educacional e promovendo a conscientização sobre sapos.
